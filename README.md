**JS Logica de Programação/Pratica 1:** 
Este projeto tem como objetivo simular o funcionamento básico de um sistema de pedidos em uma loja virtual, desenvolvido inteiramente em JavaScript. Nele, é possível escolher um produto, definir uma quantidade e obter o valor final com desconto, de acordo com a categoria do item selecionado.

O sistema utiliza um array de objetos para armazenar os produtos disponíveis, cada um contendo nome, preço, categoria e estoque. A função principal, fazerPedido(produtoEscolhido, quantidade), é responsável por localizar o produto dentro do array, verificar se ele existe e se há estoque suficiente. Caso o produto seja encontrado, o programa aplica um desconto conforme sua categoria — utilizando a estrutura switch/case — e retorna um objeto com os dados do pedido, incluindo o nome do produto, a quantidade e o preço final com desconto.

Durante o desenvolvimento, alguns ajustes foram necessários. O principal foi corrigir um erro de digitação no nome de um produto, o que impedia sua identificação. Também foi corrigido o uso incorreto do switch para validar a quantidade (que comparava valores de forma errada) e a falta de uma verificação que impedia o acesso a propriedades de um produto inexistente.

Foram aplicados diversos conceitos importantes, como funções, estruturas condicionais (if/else), método .find(), arrays, objetos e operações matemáticas básicas para cálculo de desconto.

Esse projeto é um ótimo exemplo de como combinar lógica de programação, organização de dados e estruturas de decisão em um único código funcional e bem estruturado.
